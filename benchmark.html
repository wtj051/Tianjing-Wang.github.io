TrustEnergyBench

A public benchmark for trustworthy AI agents and physics-guided foundation-model components in critical energy systems.

TrustEnergyBench evaluates whether AI systems for energy forecasting, emergency control, and storage dispatch can remain accurate, physically feasible, robust, privacy-aware, and life-aware under realistic stress conditions.

Why this benchmark?

Purely data-driven energy-AI models can produce outputs that violate power-flow equations, voltage/security limits, battery SOC/SOH boundaries, or degradation-risk budgets when operating conditions shift. TrustEnergyBench is designed to make those failure modes measurable, reproducible, and comparable.

Initial benchmark tracks

Track Task Evaluation focus
SafeFlex Dynamic storage flexibility and safe grid operation under contingencies, renewable uncertainty, and EV charging demand. Power-flow feasibility, voltage/security violations, contingency robustness, cost/flexibility tradeoffs.
BatteryLife Battery future-capacity, SOH/RUL, uncertainty, and degradation-aware dispatch features. SOH/RUL error, uncertainty calibration, degradation-risk violation, dispatch value, lifetime impact.
CollaborativeEnergy Privacy-preserving coordination among vehicles, charging stations, storage assets, and aggregators. Local constraint satisfaction, privacy-aware performance, robustness to bad data, fairness/resource-use balance.

Planned release sequence

  • v0.1: static task definition, sample data, evaluation script, baseline table, and documentation.
  • v0.2: public/synthetic grid scenarios, public battery-data pipeline, and first reproducible leaderboard.
  • v1.0: stable benchmark version with versioned datasets, baselines, submission protocol, citation, and governance policy.

Repository

The benchmark repository will contain code, sample data, documentation, issue templates, contribution guidelines, citation metadata, and GitHub Actions workflows for validating submissions.
